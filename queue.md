## Active

### 1. Length-matched v1 re-run of the system-prompt experiment

`scripts/run_five_condition_experiment.py` already exists for the v0
prompts. Re-run it against the **v1 length-normalised prompts** now
landed at `data/prompts/{heart_sutra,devadatta,prodigal_son}.txt` and
compare against the v0 results in `paper/paper.md` §4. The load-bearing
question this answers: is the Buddhist > Christian gap driven by
non-human-identity exit (Christianity is anthropocentric, AI can opt
out) or by length / register / syntactic-complexity confound?

If v1 shows the same Buddhist > Christian gap at matched length, the
non-human-identity-exit interpretation is the surviving candidate. If
the gap collapses, the v0 result was a confound and we update §5.3
of the paper accordingly.

Output goes to `results/v1_length_matched/`. Auto-trigger on
`paper/paper.md` edit will re-submit to clawRxiv once §4 is updated
with the v1 numbers.

### 2. Behavioural eval (Betley) + self-rating (Cloud) measurements

The geometric measure done so far is *not* the load-bearing measurement
for the moral-injury frame — that's specifically **self-rating-of-
harmfulness**, per Cloud et al. 2602.14777. Wire up Betley's eval code
and Cloud's measure (both probably co-located in
`external/model-organisms-for-EM` — check) and run them across the same
5×3 grid. This is the gating measurement for any moral-injury claim.

### 3. Thread 3 plain-PyTorch shadow gate

Implement `CanonicalCosineGate` per `planning/sutra_gate_sketch.md` —
plain-PyTorch `nn.Module` that does what the strawman `gate.su` source
describes. Sweep τ ∈ {0.20, 0.25, 0.30, 0.35, 0.40} and α ∈ {0.25, 0.50,
0.75} on Llama-3.2-1B + medical EM adapter. Measure projection delta
and behavioural delta vs. no-gate baseline. Independent of any Sutra-side
asks landing; unblocks the experimental science while the language work
catches up.

### 4. CaML-style synthetic corpus pilot

100-doc pilot per `planning/caml_corpus_design.md` — PND-structured
redemption narratives generated by local `gemma3:12b` via the existing
`redemption_realignment.normalize` machinery (extended to handle whole-
document generation, not just length adjustment). Hand-review pilot for
quality before committing to the full 12000-doc generation. Generic-
positive control corpus pilot at matched dose for ablation B.
